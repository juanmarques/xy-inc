# JM XY Inc Application #

API RESTful usando JAX-RS,Jersey e MySQL,
consumindo os serviços usando jQuery.

Configuração:

1. Crie um database MySql

2. Execute o arquivo product.sql para criar e popular a tabela.

3. Importe o projeto como Dynamic Web Project no Eclipse.

4. Localize o products.properties e verifique as configurações jdbc

5. Execute o projeto.
